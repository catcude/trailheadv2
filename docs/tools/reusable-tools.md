Guidepost Reusable Tools – Master List
This document outlines the reusable tools built into the Guidepost platform, divided by their core function and user experience flow. Tools are organized by where and how they appear in the app—either within adaptive dialogue or as standalone dashboard features users can access anytime.
📂 Reusable Tools – In-Dialogue Support
🧰 Micro-Needs Menu – offers subtle support prompts based on physical or emotional needs (e.g., stretch, silence, reassurance)
🌀 Mood-Matching Visual – color wheel or metaphor deck to help users name their current state
⚙️ Gentle Focus Anchor – helps refocus attention with sensory grounding or movement cues
🛠️ Covey Quadrant Sorter – helps prioritize urgent vs. important tasks (visual Covey matrix aid)
💡 Aha! Moment Tracker – lets users tag insights or clarity moments that shift motivation
🪞 Evening Wind Down Prompts – end-of-day reflection with options to preview next day, habit stack, or pause with intention
📦 Start Small Planner – helps users break big tasks into light, actionable steps
🧭 Dashboard Tools – Standalone, Habit-Building Features
✅ Habit Tracker – simple daily/weekly checklist interface for tracking self-selected habits
🔁 Streak System – gamified streak feature that reinforces consistency (with reset option and gentle messaging)
🎯 Goal Microflow Tracker – shows short-, mid-, and long-term goals and progress by category
🧠 Weekly Horizon Planner – helps students and users see the week at a glance and allocate focus by intention
📈 Progress Reflection Tools – optional check-ins for mood, progress, friction points, and course corrections
